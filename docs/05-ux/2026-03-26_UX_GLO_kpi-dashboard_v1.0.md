# 화면 정의서: 핵심 지표 대시보드

### SCR-018: KPI 대시보드 메인

#### 기본 정보

| 항목 | 내용 |

| 화면 ID | SCR-018 |
| 화면명 | KPI 대시보드 메인 |
| URL/Route | `/dashboard` |
| 인증 필요 | Y |
| 접근 권한 | 관리자, 기획자, 운영자 |

#### 화면 목적

DAU, MAU, D1 리텐션, 일 매출 4개 핵심 지표를 KPI Summary 카드로 요약하고, 회원 활동 트렌드, 리텐션 커브, 수익 트렌드 3개 영역의 차트를 단일 화면에 배치하여 운영팀의 일간 모니터링과 의사결정을 지원한다.

#### UI 요소 명세

**KpiMetricCard 시각적 규칙**

| 변화 방향 | 색상 | 아이콘 | 비고 |
|-----------|------|--------|------|
| 양수 변화 | 초록색 (text-green-600) | ▲ | 증가 |
| 음수 변화 | 빨간색 (text-red-600) | ▼ | 감소 |
| 변화 없음 | 회색 (text-gray-500) | — | 변동 없음 |

### SCR-019: 기간 토글 + 차트 인터랙션 상세

#### 기본 정보

| 항목 | 내용 |
|------|------|
| 화면 ID | SCR-019 |
| 화면명 | 기간 토글 + 차트 인터랙션 상세 |
| URL/Route | `/dashboard` (SCR-018과 동일 화면, 인터랙션 상세 정의) |
| 인증 필요 | Y |
| 접근 권한 | 관리자, 기획자, 운영자 |

#### 화면 목적

KPI 대시보드의 기간 토글(7D/30D/90D) 전환과 차트 툴팁 인터랙션의 상세 동작을 정의한다.

#### 기간 토글 상세

**토글 버튼 그룹**

```
┌──────────────────────────────────────────────────────────────┐
│  회원 활동 트렌드                         [7D] [30D] [90D]   │
│                                           ^^^^              │
│                                           기본 선택          │
└──────────────────────────────────────────────────────────────┘
```

| 기간 | 라벨 | X축 범위 | X축 포맷 | 데이터 포인트 수 |
|------|------|---------|---------|----------------|
| 7D | 7D | 오늘 ~ 7일 전 | MM/DD | 7 |
| 30D | 30D | 오늘 ~ 30일 전 | MM/DD (5일 간격) | 30 |
| 90D | 90D | 오늘 ~ 90일 전 | MM/DD (15일 간격) | 90 |

**토글 동작:**
- 기본 선택: 30D
- 선택 시: 선택된 버튼 배경색 변경 (bg-primary), 미선택 버튼 기본 스타일 유지
- API 호출: `period` 파라미터 변경 → React Query `queryKey` 변경 → 자동 refetch
- 연동 차트: 회원 활동 트렌드(E-018-05), 수익 트렌드(E-018-07) 동시 전환
- 전환 시간: 500ms 이내 차트 리렌더링 완료

**툴팁 공통 규칙:**
- 호버 시 해당 데이터 포인트에 커스텀 Recharts `<Tooltip>` 렌더링

- 텍스트: 날짜/포인트명은 볼드, 값은 오른쪽 정렬
- 숫자 포맷: 천 단위 콤마, 소수점 1자리(리텐션율), 통화 기호($)
