# 화면 정의서: 라이브 이벤트 생성 및 스케줄링

> Game LiveOps Service(GLO)의 라이브 이벤트 생성·스케줄링·승인·캘린더 기능에 대한 화면 정의서. 이벤트 전 주기(생성 → 승인 → 활성화 → 종료)를 단일 인터페이스에서 운영하는 비기술 직군 운영자를 주 대상으로 설계한다.

## 2. 화면 상세

---

### SCR-006: 이벤트 목록

#### 기본 정보

| 항목 | 내용 |
|------|------|
| 화면 ID | SCR-006 |
| 화면명 | 이벤트 목록 |
| URL/Route | `/live-events` |
| 인증 필요 | Y |
| 접근 권한 | 관리자, 기획자, 운영자 |

#### 화면 목적

라이브 이벤트 전체 목록을 조회하고 상태별 필터·검색·정렬로 원하는 이벤트를 빠르게 찾는다. 각 이벤트의 현재 상태, 스케줄, 타겟 오디언스, 우선순위를 한눈에 파악하고 생성·복제·삭제/보관 액션을 수행한다.

**행(Row) 호버 상태**

```
│ 주말 로그인 보상  [● active]  04/05~04/06  활성 플레이어  medium  03/22  [편집] [복제] [보관] │
```

> draft 상태 행의 호버 액션: `[편집] [복제] [삭제]`
> active/paused 상태 행의 호버 액션: `[상세] [복제] [보관]` (삭제 비활성화)

**상태 배지 색상 정의**

| 상태 | 라벨 | 색상 | 비고 |
|------|------|------|------|
| `draft` | draft | 회색 (#6B7280) | 수정 가능 |
| `pending_approval` | 승인 대기 | 주황 (#F59E0B) | 수정 불가 |
| `scheduled` | scheduled | 초록 (#10B981) | 시작 대기 |
| `active` | active | 파랑 (#3B82F6) | 플레이어 노출 중 |
| `paused` | paused | 노랑 (#EAB308) | 노출 일시 중단 |
| `ended` | ended | 어두운 회색 (#374151) | 종료됨 |
| `archived` | archived | 연보라 (#8B5CF6) | 읽기 전용 |

**draft 이벤트 삭제 확인 모달**

```
┌────────────────────────────────────────────────┐
│  이벤트 삭제                                    │
│                                                │
│  '봄맞이 이벤트'를 삭제합니다.                  │
│  이 작업은 되돌릴 수 없습니다.                  │
│                                                │
│                  [취소]   [삭제 확인]           │

```

**draft 외 상태 삭제 시도 차단 토스트**

```

   이벤트를 종료한 뒤 보관 처리하세요.
```

### 3.2 오디언스 셀렉터

SCR-007 이벤트 생성/편집 - 타겟팅 탭에서 사용하는 오디언스 선택 컴포넌트. UX-GLO-001 SCR-002에서 구현된 오디언스 목록을 재사용한다.

| 항목 | 내용 |
|------|------|
| 컴포넌트명 | AudienceSelector |
| 재사용 출처 | UX-GLO-001 SCR-002 (오디언스 생성 화면) 오디언스 목록 |
| 옵션 목록 | 상단 고정: "전체 유저 (audience_id NULL, 오디언스 미설정)" → 하단: 프로젝트 내 오디언스 목록 (이름 + 예상 규모) |

### 3.3 스케줄 표시 포맷

관리자 화면 전반에서 이벤트 스케줄을 일관된 형식으로 표시한다.

**단순 스케줄**

| 상황 | 표시 형식 | 예시 |
|------|----------|------|
| 상세 표시 | `YYYY-MM-DD HH:mm ~ YYYY-MM-DD HH:mm (TZ)` | `2026-04-05 09:00 ~ 2026-04-06 23:59 (KST)` |
| 목록 축약 | `MM/DD ~ MM/DD` | `04/05 ~ 04/06` |
| 종료 없음 | `YYYY-MM-DD HH:mm ~ (종료 미정) (TZ)` | `2026-04-05 09:00 ~ (종료 미정) (KST)` |

**반복 스케줄 (rrule 한국어 설명)**

| rrule 예시 | 한국어 설명 |
|-----------|------------|
| `FREQ=WEEKLY;BYDAY=FR` + duration 1440분 | 매주 금요일 00:00, 24시간 (KST) |
| `FREQ=MONTHLY;BYMONTHDAY=1` + duration 1440분 | 매월 1일 00:00, 24시간 (KST) |
| `FREQ=DAILY` + duration 60분 | 매일 00:00, 60분 (KST) |

- display_timezone이 설정된 경우 해당 타임존 약어 표시 (예: KST, PST). 미설정 시 UTC 표시
- 목록 화면(SCR-006)에서 반복 스케줄은 "반복 (매주 금요일)" 형식으로 요약 표시
