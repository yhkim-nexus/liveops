# 화면 정의서: 역할 기반 접근 제어 (RBAC)

> Game LiveOps Service(GLO)의 역할 기반 접근 제어(RBAC) 기능에 대한 화면 정의서. Admin/Operator/Viewer 3역할 기반으로 로그인·GNB 네비게이션 필터링·관리자 관리·감사 로그·권한 없음 화면을 정의한다. 비기술 직군 운영자를 주 대상으로 설계한다.

## 1. Information Architecture

### 1.1 사이트맵 (RBAC 영역)

```
[관리자 대시보드]
├── 로그인 (SCR-020)                    ← 공개
├── GNB (SCR-021)                       ← 전 화면 공통
├── 대시보드 (/dashboard)                ← viewer 이상
├── 이벤트 (/events/*)                   ← operator 이상
├── 실험 (/experiments/*)                ← operator 이상
├── 세그먼트 (/segments/*)               ← operator 이상
├── 설정 (Settings)                      ← admin 전용
│   ├── 관리자 목록 (SCR-022)
│   ├── 관리자 초대 (SCR-023)
│   └── 감사 로그 (SCR-024)
└── 403 페이지 (SCR-025)                ← 권한 부족 시
```

### 1.2 역할별 네비게이션 필터링

| 메뉴 | 경로 | viewer | operator | admin |
|------|------|--------|----------|-------|
| 대시보드 | `/dashboard` | ✅ | ✅ | ✅ |
| 이벤트 | `/events` | ❌ | ✅ | ✅ |
| 실험 | `/experiments` | ❌ | ✅ | ✅ |
| 세그먼트 | `/segments` | ❌ | ✅ | ✅ |
| 설정 | `/settings` | ❌ | ❌ | ✅ |

> Viewer는 대시보드만 접근 가능, Operator는 대시보드 + 이벤트/실험/세그먼트, Admin은 전체 메뉴 접근 가능

### 1.3 화면-기능 매핑

| 화면 ID | 화면명 | 주요 기능 |
|---------|--------|----------|
| SCR-020 | 로그인 페이지 | F-028(Mock 인증: 로그인) |
| SCR-021 | GNB | F-033(역할 기반 네비게이션), F-028(로그아웃) |
| SCR-022 | 관리자 목록 | F-031(관리자 관리: 목록/역할변경), F-029(역할 모델) |
| SCR-023 | 관리자 초대 | F-031(관리자 관리: 초대) |
| SCR-024 | 감사 로그 | F-032(감사 로그 조회) |
| SCR-025 | 403 페이지 | F-030(Middleware 라우트 보호: 권한 부족 안내) |

### SCR-020: 로그인 페이지

#### 기본 정보

| 항목 | 내용 |
|------|------|
| 화면 ID | SCR-020 |
| 화면명 | 로그인 페이지 |
| URL/Route | `/login` |
| 인증 필요 | N (공개) |
| 접근 권한 | 없음 |

#### 화면 목적

관리자가 이메일과 비밀번호를 입력하여 LiveOps 대시보드에 인증하는 진입 화면. 인증 성공 시 `/dashboard`로 리다이렉트하며, 이미 인증된 사용자는 자동으로 `/dashboard`로 이동한다.

#### UI 요소 명세

| ID | 요소명 | 타입 | 필수 | 설명 |
|----|--------|------|------|------|
| E-020-01 | 이메일 입력 | Input (email) | Y | 이메일 형식 검증 (Zod). placeholder: "이메일을 입력하세요" |
| E-020-02 | 비밀번호 입력 | Input (password) | Y | 최소 1자. placeholder: "비밀번호를 입력하세요" |

#### 인터랙션 명세

| ID | 트리거 | 액션 | 결과 |
|----|--------|------|------|

| INT-020-02 | 이미 인증된 상태로 `/login` 접근 | 세션 쿠키 확인 | `/dashboard`로 자동 리다이렉트 |

### SCR-021: GNB (Global Navigation Bar)

#### 기본 정보

| 항목 | 내용 |
|------|------|
| 화면 ID | SCR-021 |
| 화면명 | GNB (Global Navigation Bar) |
| URL/Route | 전 화면 공통 (인증 후) |
| 인증 필요 | Y |
| 접근 권한 | viewer 이상 |

#### 화면 목적

인증된 관리자에게 역할에 따라 필터링된 네비게이션 메뉴를 제공한다. 좌측에 로고와 메뉴 링크, 우측에 관리자 정보(이름 + 역할 배지)와 로그아웃 버튼을 배치한다.

#### 와이어프레임

**Admin 역할 (전체 메뉴)**

```
┌─────────────────────────────────────────────────────────────────────┐
│  [GLO]  대시보드  이벤트  실험  세그먼트  설정       김관리 [Admin] [로그아웃] │
└─────────────────────────────────────────────────────────────────────┘
   E-021-01  ───── E-021-02 ─────────────    E-021-03 E-021-04 E-021-05
```

**Operator 역할 (설정 메뉴 미표시)**

```
┌─────────────────────────────────────────────────────────────────────┐
│  [GLO]  대시보드  이벤트  실험  세그먼트               이운영 [Operator] [로그아웃] │
└─────────────────────────────────────────────────────────────────────┘
```

**Viewer 역할 (대시보드만)**

```
┌─────────────────────────────────────────────────────────────────────┐
│  [GLO]  대시보드                                     박조회 [Viewer] [로그아웃] │
└─────────────────────────────────────────────────────────────────────┘
```

#### UI 요소 명세

| ID | 요소명 | 타입 | 필수 | 설명 |
|----|--------|------|------|------|
| E-021-01 | 로고 | Link | Y | 클릭 시 `/dashboard`로 이동 |
| E-021-02 | 네비게이션 링크 | NavLink 그룹 | Y | 역할 기반 필터링. 현재 활성 메뉴 하이라이트 |
| E-021-03 | 관리자 이름 | Text | Y | 현재 로그인한 관리자 이름 표시 |
| E-021-04 | 역할 배지 | Badge | Y | Admin: 빨강, Operator: 파랑, Viewer: 회색 |
| E-021-05 | 로그아웃 버튼 | Button (ghost) | Y | 클릭 시 POST /api/auth/logout → `/login` 리다이렉트 |

#### 인터랙션 명세

| ID | 트리거 | 액션 | 결과 |
|----|--------|------|------|
| INT-021-01 | 네비게이션 링크 클릭 | 해당 경로로 이동 | 역할 권한 부족 시 Middleware에서 403 처리 |
| INT-021-02 | 로그아웃 버튼 클릭 | POST /api/auth/logout | 세션 삭제 → `/login` 리다이렉트 + 감사 로그 기록 |

### SCR-023: 관리자 초대

#### 기본 정보

| 항목 | 내용 |
|------|------|
| 화면 ID | SCR-023 |
| 화면명 | 관리자 초대 |
| URL/Route | `/settings/admins/invite` |
| 인증 필요 | Y |
| 접근 권한 | admin |

#### 화면 목적

신규 관리자를 초대(생성)한다. 이메일, 이름, 역할을 입력하여 관리자 계정을 생성하고, 성공 시 관리자 목록(SCR-022)으로 리다이렉트한다.

#### 와이어프레임

**Default State**

```
┌─────────────────────────────────────────────────────────────────────┐
│  [GLO]  대시보드  이벤트  실험  세그먼트  [설정]     김관리 [Admin] [로그아웃] │
```

```
│               │          🔒                       │                 │
│               │                                  │                 │
│               │   접근 권한이 없습니다              │  E-025-01       │

│                                                                     │
```

#### UI 요소 명세

| ID | 요소명 | 타입 | 필수 | 설명 |
|----|--------|------|------|------|
